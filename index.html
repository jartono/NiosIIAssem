This is a visualization-only UI to help you see how a 32-bit NIOS II instruction is split into fields. In a next step we can wire this up to actually decode and encode instructions from assembly mnemonics and register names into machine code and back.

SyntaxDescription

nopNo operation (add r0, r0, r0)
not rD, rSBitwise NOT (nor rD, rS, r0)
